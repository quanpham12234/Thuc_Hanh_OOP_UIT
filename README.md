# Thuc_Hanh_OOP_UIT

## 📌 Introduction

This repository contains practical exercises and assignments for **Object-Oriented Programming (OOP)** at UIT, implemented in **C++**.

The goal is to strengthen understanding of core OOP concepts through coding practice and problem solving.

---

## 🎯 Topics Covered

* Classes and Objects
* Constructors / Destructors
* Encapsulation
* Inheritance
* Polymorphism
* Abstraction
* Operator Overloading
* File Handling
* Mini Projects
